# SurvLingo

## Overview

SurvLingo is the AI intelligence platform powering the Survonomics ecosystem.

Purpose-built for the construction industry, SurvLingo helps professionals communicate, understand technical documents, estimate costs, monitor budgets, and make better project decisions.

Rather than acting as a generic chatbot, SurvLingo serves as an intelligent construction co-pilot that works alongside project teams throughout the entire project lifecycle.

It integrates seamlessly with:

- SurvMessenger
- SurvPay
- SurvPrice
- SurvMarket

creating one intelligent ecosystem for construction professionals.
